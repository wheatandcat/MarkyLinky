# Product Overview

## プロダクト概要
Chrome 拡張「MarkyLinky」。閲覧ページをワンクリックで保存・削除し、Markdown 形式で共有できるブックマーク管理拡張。Supabase 認証と連携したクラウド同期に対応。

## 主要機能
- ポップアップから現在タブの URL を保存/削除
- 保存一覧の検索・Markdown コピー（`[title](url)`）
- ダーク/ライト切替と favicons 表示
- Supabase OAuth ログイン（GitHub/Google）とデータ同期
- コンテキストメニューからの追加/削除トグル

## 対象ユースケース
調査・執筆時のリンク収集、読み返し用の簡易ブックマーク、Markdown ベースのノートや Issue への貼り付け。

## 価値提案
ブラウザ内で完結する軽量 UI と、ログイン時は Supabase に自動同期されるシンプルな二層保存（ローカル + クラウド）。Markdown ワンコピーでドキュメント共有が容易。
